# Python Notes

## Basics and Fundametals

Python is a High Level, Object-Oriented, Dynamically Typed, Interpreted Language. It is Simple and Easy to learn and has extensive standard library. 

Some other Properties of Python - 
* Open Source and Free
* Portable
* Extensible and Embeddable
* Automatic Memory management
* Large Ecosystem

## Data Types in Python

* Text Data Type - String
* Number Data Type - Int, float, complex
* Sequence Data Type - list, tuple, range
* Set Data Type - set, frozenset
* Mapping Data Type - dict
* Boolean Data Type - bool
* Binary Data Type - bytes, bytearray, memoryview
* None Data Type - NoneType
